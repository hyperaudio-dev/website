Icons.

Icons rely on an icon font located in assets/fonts/hyperaudio. The font was custom build using IcoMoon app.

To extend the font with additional characters, import selection.json within the app, add in needed characters and export the font with “hyperaudio” as name and “icon-” as prefix.

Placing an icon on a page is as simple as add in a i with appropriate class, for example:

<i class="icon-logo-icon-logo-bbc"></i>
<i class="icon-icon-logo-aljazeera"></i>
<i class="icon-logo-mozilla"></i>
