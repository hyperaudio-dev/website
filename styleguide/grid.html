Spaces.

A set of spacing variables is in place to keep paddings and margins consistent throughout the Hyperaudio interaface. Whenever adjusting elements’ box model, please use the following values defined in the _config.scss file:

$space-h
$space-xl
$space-l
$space-m
$space-s
$space-xs

Bad:

.element { padding: 5px }

Good:

.element { padding: $space-s }
